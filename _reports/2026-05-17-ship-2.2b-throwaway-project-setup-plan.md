## Task 2: Build the JSON-backup loader `restore_supabase.py`

**Does this plan need a loader? Yes.** The backup is seven `<table>.json` files
(JSON arrays of row objects) â€” `dump_supabase.py`'s output format. A fresh
Supabase project starts empty; there is no Supabase-native "import a JSON
backup" path, and the Supabase CLI's restore works on `pg_dump` SQL, not JSON.
So a loader is required. It belongs in **`venture-ops/`**, beside
`dump_supabase.py`, as its deliberate inverse â€” studio infrastructure, **never**
copied into the `tidy-tails` repo.

Built TDD: the pure helpers (key-role decode, content-range parse, batching,
SHA-256, FK ordering, redaction) are unit-tested with the Python standard

  ```python
  #!/usr/bin/env python3
  """restore_supabase.py â€” load a dump_supabase.py backup into a TARGET project.

  Studio restore tool. The inverse of dump_supabase.py: reads a backup directory
  (MANIFEST.json + one <table>.json per table) and INSERTs every row into a
  *target* Supabase project through the REST API (PostgREST).

  Built for the Tidy Tails v2 Ship 2.2b RLS rehearsal â€” it populates a throwaway
  Supabase project with production's data so the RLS migration can be rehearsed
  against a full-fidelity copy. See
  `tidy-tails/_reports/2026-05-17-ship-2.2b-throwaway-project-setup-plan.md`.

  WHAT IT DOES
    - Verifies each <table>.json SHA-256 against the manifest before loading â€”
      a corrupted or altered backup file fails the run, never loads silently.
    - Loads tables in FK-dependency order (parents before children) so foreign
      keys resolve: clients -> pets -> appointments -> ... .
    - Refuses to load into a table that is not already empty (prevents
      double-loading and refuses production, whose tables are populated).
    - Inserts rows verbatim, including their primary keys, in batches via
      PostgREST POST with `Prefer: return=minimal`.
    - Reconciles, per table, the post-load exact count against the manifest's
      row_count. A mismatch fails the run.

  WHAT IT DOES NOT DO (by design)
    - It never targets production. It writes only to the project named by
      SUPABASE_URL â€” point that at the throwaway rehearsal target, never prod.
      A typed-confirmation prompt guards this unless --yes is passed.
    - It does not create schema. The target's tables, RLS, policies, and the
      client_overview view must already exist (setup plan, Task 5).
    - It does not apply the RLS migration, create auth users, or run DDL.
    - It does not commit anything.

  CREDENTIALS  (never hardcoded â€” read from the environment)
    SUPABASE_URL   the TARGET project URL, e.g. https://<ref>.supabase.co
    SUPABASE_KEY   the TARGET project's service_role key (a JWT: "eyJ...").

    Use the SERVICE_ROLE key: INSERTs must bypass Row Level Security so every
    row lands regardless of the target's (still permissive) policy state. The
    script refuses a non-service_role key unless --allow-non-service-role.
    The key value is NEVER written anywhere: not stdout, not stderr, not a file.

  USAGE
    export SUPABASE_URL='https://<throwaway-ref>.supabase.co'
    export SUPABASE_KEY='<target service_role key>'
    python3 restore_supabase.py \\
      --backup-dir backups/tidy-tails/2026-05-18-rehearsal

  EXIT CODES
    0  every table loaded and reconciled
    1  ran but a table failed (sha mismatch, non-empty target, count mismatch)
    2  precondition failed (missing env var, refused key role, bad usage)
  """

  from __future__ import annotations

  import argparse
  import base64
  import hashlib
  import json
  import os
  import sys
  import urllib.error
  import urllib.request

  SCRIPT_VERSION = "1.0.0"
  HTTP_TIMEOUT = 60  # seconds per request

  # FK-dependency order: a parent table is always loaded before any table that
  # references it. clients has no FK; pets.client_id -> clients; appointments
  # references clients and pets. The three empty tables and sam_review_responses
  # carry no rows referencing the 6-table set in this backup, but the order is
  # kept FK-safe regardless. Confirm against the Task 1 schema dump before a run.
  LOAD_ORDER = [
      "clients",
      "pets",
      "appointments",
      "booking_requests",
      "client_accounts",
      "automations_log",
      "sam_review_responses",
  ]


  # ----------------------------------------------------------------------
  # Pure helpers (no I/O) â€” unit-tested in test_restore_supabase.py.
  # ----------------------------------------------------------------------

  def decode_key_role(key: str) -> str:
      """Return the `role` claim from a Supabase API key (a JWT).

      Decodes the JWT payload only â€” no signature check; the role claim is not
      a secret and is used solely for the safety gate. Raises ValueError if the
      key is not a decodable JWT with a role claim.
      """
      parts = key.split(".")
      if len(parts) != 3:
          raise ValueError(
              "SUPABASE_KEY is not a JWT (expected three dot-separated parts)."
          )
      segment = parts[1]
      segment += "=" * (-len(segment) % 4)  # restore base64url padding
      try:
          payload = json.loads(base64.urlsafe_b64decode(segment))
      except Exception as exc:  # noqa: BLE001 â€” any decode failure is fatal
          raise ValueError("Could not decode the SUPABASE_KEY JWT payload.") \
              from exc
      role = payload.get("role")
      if not role:
          raise ValueError("SUPABASE_KEY JWT payload has no 'role' claim.")
      return str(role)


  def parse_content_range_total(header: str | None) -> int:
      """Extract the exact total from a PostgREST `Content-Range` header.

      Formats: "0-49/737" (populated) and "*/0" (empty table). A "*" total
      means the server did not count â€” treated as an error, never guessed.
      """
      if not header or "/" not in header:
          raise ValueError(
              f"Missing or malformed Content-Range header: {header!r}")
      total = header.split("/")[-1].strip()
      if total == "*":
          raise ValueError("Server returned an unknown total (Content-Range).")
      return int(total)


  def chunked(seq: list, size: int) -> "list[list]":
      """Split a list into consecutive batches of at most `size` items."""
      if size < 1:
          raise ValueError("batch size must be >= 1")
      return [seq[i:i + size] for i in range(0, len(seq), size)]


  def sha256_of_file(path: str) -> str:
      """Return the SHA-256 hex digest of a file's bytes."""
      digest = hashlib.sha256()
      with open(path, "rb") as handle:
          for block in iter(lambda: handle.read(65536), b""):
              digest.update(block)
      return digest.hexdigest()


  def order_tables(manifest_tables: "list[str]",
                   load_order: "list[str]") -> "list[str]":
      """Return manifest tables sorted into FK-safe load order.

      Raises ValueError if the manifest holds a table not in load_order â€” an
      unknown table could be a child whose parent is not sequenced, so the run
      stops rather than guessing.
      """
      unknown = [t for t in manifest_tables if t not in load_order]
      if unknown:
          raise ValueError(
              f"manifest tables not in LOAD_ORDER: {sorted(unknown)}")
      return [t for t in load_order if t in manifest_tables]


  def redact(text: str, secret: str) -> str:
      """Remove a secret from a string before it is printed."""
      return text.replace(secret, "***REDACTED***") if secret else text


  # ----------------------------------------------------------------------
  # HTTP
  # ----------------------------------------------------------------------

  def _request(url: str, key: str, method: str, extra_headers: dict,
               data: bytes | None = None) -> "tuple[dict, bytes]":
      """Issue one authenticated request. Returns (response headers, body)."""
      request = urllib.request.Request(url, data=data, method=method)
      request.add_header("apikey", key)
      request.add_header("Authorization", f"Bearer {key}")
      if data is not None:
          request.add_header("Content-Type", "application/json")
      for name, value in extra_headers.items():
          request.add_header(name, value)
      with urllib.request.urlopen(request, timeout=HTTP_TIMEOUT) as response:
          return dict(response.headers), response.read()


  def target_count(base_url: str, key: str, table: str) -> int:
      """Return the target's exact row count for one table (count=exact)."""
      url = f"{base_url}/rest/v1/{table}?select=id"
      headers, _ = _request(
          url, key, "GET",
          {"Range-Unit": "items", "Range": "0-0", "Prefer": "count=exact"},
      )
      return parse_content_range_total(headers.get("Content-Range"))


  def insert_rows(base_url: str, key: str, table: str, rows: list,
                  batch_size: int) -> int:
      """POST every row of one table in batches. Returns the count inserted."""
      url = f"{base_url}/rest/v1/{table}"
      inserted = 0
      for batch in chunked(rows, batch_size):
          body = json.dumps(batch).encode("utf-8")
          _request(url, key, "POST", {"Prefer": "return=minimal"}, data=body)
          inserted += len(batch)
      return inserted


  # ----------------------------------------------------------------------
  # Main
  # ----------------------------------------------------------------------

  def main(argv: list[str]) -> int:
      parser = argparse.ArgumentParser(
          description="Load a dump_supabase.py backup into a target project.",
          formatter_class=argparse.RawDescriptionHelpFormatter,
      )
      parser.add_argument(
          "--backup-dir", required=True,
          help="A dump_supabase.py backup directory (holds MANIFEST.json).")
      parser.add_argument(
          "--batch-size", type=int, default=500,
          help="Rows per POST request. Default 500.")
      parser.add_argument(
          "--allow-non-service-role", action="store_true",
          help="Permit a non-service_role key (inserts may be RLS-blocked).")
      parser.add_argument(
          "--yes", action="store_true",
          help="Skip the typed target-confirmation prompt.")
      args = parser.parse_args(argv)

      if args.batch_size < 1:
          print("error: --batch-size must be >= 1", file=sys.stderr)
          return 2

      base_url = (os.environ.get("SUPABASE_URL") or "").rstrip("/")
      key = os.environ.get("SUPABASE_KEY") or ""
      if not base_url or not key:
          print("error: set SUPABASE_URL and SUPABASE_KEY in the environment.",
                file=sys.stderr)
          return 2

      # --- Key-role safety gate. ---
      try:
          key_role = decode_key_role(key)
      except ValueError as exc:
          if not args.allow_non_service_role:
              print(f"error: {exc}\n  Re-run with the service_role key, or "
                    "pass --allow-non-service-role.", file=sys.stderr)
              return 2
          key_role = "unknown"
      if key_role != "service_role" and not args.allow_non_service_role:
          print(f"error: SUPABASE_KEY role is '{key_role}', not 'service_role'."
                "\n  INSERTs may be silently RLS-blocked. Re-run with the "
                "service_role key.", file=sys.stderr)
          return 2

      # --- Manifest. ---
      manifest_path = os.path.join(args.backup_dir, "MANIFEST.json")
      try:
          with open(manifest_path, encoding="utf-8") as handle:
              manifest = json.load(handle)
      except (OSError, ValueError) as exc:
          print(f"error: cannot read {manifest_path}: {redact(str(exc), key)}",
                file=sys.stderr)
          return 2
      if manifest.get("status") != "complete":
          print(f"error: manifest status is {manifest.get('status')!r}, not "
                "'complete' â€” refusing to load an incomplete backup.",
                file=sys.stderr)
          return 2

      try:
          tables = order_tables(list(manifest.get("tables", {})), LOAD_ORDER)
      except ValueError as exc:
          print(f"error: {exc}", file=sys.stderr)
          return 2

      ref = base_url.split("//")[-1].split(".")[0] if "//" in base_url \
          else base_url
      print(f"Target project: {base_url}  (ref: {ref}, key role: {key_role})")
      print(f"Backup: {args.backup_dir}")

      # --- Typed confirmation: do NOT load into production by mistake. ---
      if not args.yes:
          typed = input(
              f"Type the target ref ({ref}) to confirm this is the throwaway "
              "rehearsal project, NOT production: ").strip()
          if typed != ref:
              print("error: confirmation did not match â€” aborting.",
                    file=sys.stderr)
              return 2

      # --- Load each table in FK order. ---
      failed = False
      for table in tables:
          entry = manifest["tables"][table]
          file_path = os.path.join(args.backup_dir, entry["file"])

          actual_sha = sha256_of_file(file_path)
          if actual_sha != entry["sha256"]:
              failed = True
              print(f"  FAIL {table}: sha256 mismatch â€” backup file altered.",
                    file=sys.stderr)
              continue

          with open(file_path, encoding="utf-8") as handle:
              rows = json.load(handle)
          if len(rows) != entry["row_count"]:
              failed = True
              print(f"  FAIL {table}: file has {len(rows)} rows, manifest "
                    f"says {entry['row_count']}.", file=sys.stderr)
              continue

          try:
              before = target_count(base_url, key, table)
          except (urllib.error.URLError, ValueError) as exc:
              failed = True
              print(f"  FAIL {table}: cannot read target count: "
                    f"{redact(str(exc), key)}", file=sys.stderr)
              continue
          if before != 0:
              failed = True
              print(f"  FAIL {table}: target already has {before} rows â€” "
                    "refusing to load.", file=sys.stderr)
              continue

          try:
              if rows:
                  insert_rows(base_url, key, table, rows, args.batch_size)
              after = target_count(base_url, key, table)
          except (urllib.error.URLError, ValueError) as exc:
              failed = True
              print(f"  FAIL {table}: {redact(str(exc), key)}",
                    file=sys.stderr)
              continue
          if after != entry["row_count"]:
              failed = True
              print(f"  FAIL {table}: loaded but target count is {after}, "
                    f"expected {entry['row_count']}.", file=sys.stderr)
              continue
          print(f"  ok   {table}: {after} rows loaded and reconciled")

      if failed:
          print("error: restore did not complete cleanly â€” see above.",
                file=sys.stderr)
          return 1
      print("Restore complete â€” every table loaded and reconciled.")
      return 0


  if __name__ == "__main__":
      try:
          sys.exit(main(sys.argv[1:]))
      except KeyboardInterrupt:
          sys.exit(130)
  ```
